# 🎬 Movie Recommendation System (End-to-End)

## 📖 Overview

This project is an **end-to-end Movie Recommendation System** built using **Python, Machine Learning, API integration, and a web application**.

It recommends movies to users based on similarity and user preferences using content-based filtering techniques.

---

## 🚀 Features

* 🎥 Movie recommendation based on similarity
* 🌐 API integration to fetch movie posters/details
* 📊 Exploratory Data Analysis (EDA)
* 🤖 Machine Learning model for recommendations
* 💻 Web application built using Python (PyCharm)

---

## 🛠️ Tech Stack

* **Python** (Pandas, NumPy, Scikit-learn)
* **Machine Learning** (Content-Based Filtering)
* **API** (for movie data & posters)
* **Flask / Streamlit** (Web App)
* **Git & GitHub**

---

## 🔄 Project Workflow

1. **Data Collection**

   * Movie dataset (titles, genres, keywords)

2. **Data Preprocessing**

   * Handling missing values
   * Feature extraction

3. **EDA (Exploratory Data Analysis)**

   * Genre distribution
   * Popular movies analysis
   * Trends and patterns

4. **Feature Engineering**

   * Combined features (genres, cast, keywords)
   * Text vectorization (TF-IDF / CountVectorizer)

5. **Model Building**

   * Cosine similarity used for recommendations

6. **API Integration**

   * Fetch movie posters dynamically

7. **Web Application**

   * User inputs movie name
   * Displays recommended movies with posters

* System finds similar movies using cosine similarity
* API fetches posters
* Results displayed on web interface
